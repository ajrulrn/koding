Atribut `value` pada sebuah input adalah atribut yang berguna untuk memberi nilai awal pada sebuah input.


## Contoh penggunaan atribut value

```html
<input type='text' value="username">
```
Hasilnya

{{<rawhtml>}}
<input type='text' value="username">
{{</rawhtml>}}
